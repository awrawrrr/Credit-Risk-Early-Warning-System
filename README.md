**Objective**

Develop an Early Warning Indicator framework to forecast Non Performing Financing (NPF) and identify customers with elevated default risk prior to write-off events.



**Methodology**

* Data Extraction \& Validation
* Feature Engineering
* ARX (AutoRegressive with Exogenous Variables) Model
* Model Evaluation using RMSE



**Key Results**

* Developed a time series based ARX model to capture dynamics of NPF behavior across portfolio segments.
* Achieved stable forecasting performance as measured by RMSE, indicating reliable short term predictive capability.
* Identified key lagged and exogenous variables that significantly influence NPF movement across time.



**Model Output** 

The model performance across DH segments shows relatively stable predictive accuracy, with RMSE ranging from 6.69% to 13.74%. DH 4 achieved the best performance, indicating strong model fit, while DH 2 exhibited higher forecasting error, suggesting greater volatility in that segment.



**Business Impact**

The framework enables proactive monitoring of potentially delinquent customers prior to write off realization, allowing early intervention strategies to mitigate credit risk exposure.
